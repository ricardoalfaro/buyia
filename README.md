# Buyia - Landing Page

Landing page moderna y de alto impacto para **Buyia**, un asistente de ventas por WhatsApp impulsado por IA.

## Descripción del Proyecto

Esta página ha sido diseñada para destacar la simplicidad y el poder de conversar para vender. Utiliza una estética limpia, con mucho espacio en blanco, tonos azules suaves y acentos en verde lima/aqua para transmitir confianza y frescura.

### Características Principales

*   **Diseño Responsivo:** Adaptado perfectamente para dispositivos móviles (Mobile First).
*   **Chat UI con CSS Puro:** Los mockups de las conversaciones de WhatsApp están construidos con HTML y CSS para asegurar nitidez y tiempos de carga rápidos.
*   **Animaciones Sutiles:** Transiciones suaves en hover y una simulación de chat en vivo en la sección de "Demo".
*   **Paleta de Colores Personalizada:** Configuración de Tailwind extendida para coincidir con la marca (Soft Blue, Aqua, Lime).

## Estructura del Sitio

1.  **Hero Section:** Título impactante, mockups de chat y CTAs claros.
2.  **Beneficios:** Grid de tarjetas destacando las ventajas clave (Conversión, 24/7, IA).
3.  **Cómo Funciona:** Proceso visual de 4 pasos.
4.  **Casos de Uso:** Sectores donde Buyia es ideal.
5.  **Demo en Vivo:** Simulación automática de una conversación de venta.
6.  **Filosofía:** "No navegas. Conversas."
7.  **Footer:** Enlaces y derechos de autor.

## Tecnologías Utilizadas

*   **HTML5:** Semántico y accesible.
*   **Tailwind CSS (CDN):** Para el estilizado rápido y consistente.
*   **JavaScript (Vanilla):** Para la lógica del menú móvil y la animación del chat demo.
*   **Font Awesome:** Para la iconografía.
*   **Google Fonts:** Tipografía 'Inter' para un look moderno y limpio.

## Cómo Editar

El archivo principal es `index.html`. 
*   La configuración de colores se encuentra en el script de configuración de Tailwind en el `<head>`.
*   Los textos de la demo del chat se pueden modificar en el array `demoMessages` dentro del script al final del `body`.

## Despliegue

Para publicar este sitio, simplemente ve a la pestaña **Publish** y haz clic en desplegar. No se requiere configuración adicional del servidor.
